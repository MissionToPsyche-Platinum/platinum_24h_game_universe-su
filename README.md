# Psyche Against the Universe

A web-based game designed to build public interest in NASA's Psyche mission.

## About

**Psyche Against the Universe** is an educational web game that combines engaging gameplay with accurate information about NASA's Psyche mission. The game features details about the mission, including spacecraft features, mission timing, and the teams involved.

## Game Details

- **Playtime**: Approximately 30 minutes
- **Target Audience**: All ages - from young players to adults interested in space
- **Gameplay Style**: Cartoony/realistic mechanics designed to be approachable and accessible
- **Educational Value**: Includes genuine facts about the Psyche mission

## Purpose

By combining accurate details about the Psyche mission with creative challenges and playful design, the game invites players to imagine themselves as part of this NASA mission.

## Technology

Built with Unity for web deployment.
